Templates Mailing List

Top Close Open

The templates mailing list exists for reporting information, asking questions, discussing development or any other topic relevant to the Template Toolkit.

To subscribe, send an email to templates-request@template-toolkit.org with the word 'subscribe' in the body or use the web form here:

http://mail.template-toolkit.org/mailman/listinfo/templates

You can read the archive of previous posts here:

http://mail.template-toolkit.org/pipermail/templates/

Templates-Announce Mailing List

Top Close Open

The templates-announce mailing list is a low-volume list used for announcing new versions of the Template Toolkit and other related information.

To subscribe, send an email to templates-announce-request@template-toolkit.org with the word 'subscribe' in the body or use the web form here:

http://mail.template-toolkit.org/mailman/listinfo/templates-announce

You can read the archive of previous posts here:

http://mail.template-toolkit.org/pipermail/templates-announce/

Templates-SVN Mailing List

Top Close Open

The templates-svn mailing list is a read-only list for developers who want to keep track of updates in the Subversion source code repository for the Template Toolkit. Each commit to the repository is forwarded to the list.

Subscribing is the same as for the other lists. Either send an email to templates-svn-request@template-toolkit.org with the word 'subscribe' in the body or use the web form here:

http://mail.template-toolkit.org/mailman/listinfo/templates-svn

You can read the archive of previous posts here:

http://mail.template-toolkit.org/pipermail/templates-svn/

We switched to using Subversion early in 2007. Before that we used CVS and had a corresponding templates-cvs mailing list. You can read the archive of templates-cvs posts here:

http://mail.template-toolkit.org/pipermail/templates-cvs/
